Weather App

This is a responsive weather forecasting web application built using React.js. It fetches real-time weather data using the OpenWeatherMap API. Users can search for any city worldwide and view updated weather conditions. The app displays temperature, humidity, wind speed, and weather icons. It also shows the latitude and longitude of the searched location. Icons dynamically change based on weather conditions using custom image assets. The app handles loading states and invalid city names gracefully. This project showcases my skills in API integration, state management, and responsive UI design.
